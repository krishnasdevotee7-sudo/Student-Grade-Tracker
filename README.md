# Student Grade Tracker

## What This Does
A program that tracks student grades, calculates averages, and assigns letter grades.

## Features
- Display all students with grades
- Find highest and lowest scores
- Calculate class average
- Count students by grade

## How to Run
1. Open main.py
2. Run the program
3. See the output

## Skills Used
- Python loops
- Conditionals (if/elif/else)
- Data structures (lists, tuples)
- F-strings

## Example Output
Alice - 95 - Grade A
Bob - 87 - Grade B
